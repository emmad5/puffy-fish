# Puffy Fish README
[Live Game](https://emmad5.github.io/puffy-fish/)

## Overview
Similar to flappy bird, puffy fish is a simple one player game where the main character, a puffer fish, has to jump to avoid seaweed obstacles. The fish canâ€™t jump out of the water and has to avoid seaweed below. There is also a fishing line from above that the fish must avoid. The user presses the spacebar to swim similar to flappy bird.
